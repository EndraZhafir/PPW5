Contoh Macam Fungsi CSS

1. text-indent

Contoh dari text-indent. Mengatur indentasi teks pada awal paragraf. Cocok untuk membuat tampilan paragraf lebih rapi.

2. letter-spacing

Contoh dari letter-spacing. Mengatur jarak antara huruf dalam teks. Dapat digunakan untuk meningkatkan keterbacaan atau untuk efek estetika.

3. word-spacing

Contoh dari word-spacing. Mengatur jarak antara kata-kata dalam teks. Berguna untuk memperjelas pemisahan antar kata.

4. line-height

Contoh dari line-height. Mengatur tinggi baris dalam teks. Ini berpengaruh pada jarak vertikal antara baris teks.

5. white-space

Contoh dari white-space. Mengontrol bagaimana spasi dan garis baru ditangani dalam teks. Dapat digunakan untuk mempertahankan spasi tambahan atau mencegah pemotongan kata.
